Anime.js 官網

關於Anime.js的小介紹

是一個輕量且強大的 JavaScript 動畫庫，專為網頁動畫設計，適合製作視覺效果豐富的動畫。它的優勢在於簡單易用，能夠操控 HTML 元素、CSS 屬性、SVG、DOM 屬性以及 JavaScript 物件等多種元素。

Anime.js 核心概念與功能介紹

  • 多屬性動畫： 可以同時操控多個屬性，如顏色、位置、尺寸等。
  • 緩動效果： 提供多種緩動函數，使動畫更平滑自然。
  • 時間線： 可設置不同動畫的時間線順序，簡化複雜動畫效果。
  • 迭代與回圈： 可設置動畫的重複次數，或設定無限迴圈。
  • 回調函數： 動畫開始、完成、重複時都可以添加回調函數。
